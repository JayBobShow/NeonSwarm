# Neon Swarm Story Bible

## Phase 40 Naming Lock

This document is the official story foundation for Neon Swarm as of Phase 40.
It locks the world, hero identity, main threat, sector story roles, story terms,
opening outline, and ending outline before any cinematic implementation begins.

Phase 40 does not implement an opening cutscene, ending cutscene, cinematic
system, new gameplay system, new enemy behavior, or new boss behavior.

## Core Identity

- Game title: Neon Swarm
- Tagline: One light. Endless swarm. A universe worth remembering.
- Hero: Nova Veyr
- Player form: The Aether Core
- Guide / engineer: Lyra Quill
- Old machine ally: Rook-7
- Lost friend / emotional core: Mira Sol
- Main villain: The Null King
- Main threat: The Swarm
- World: The Neon Grid
- Main story theme: memory, survival, color, motion, and refusing to let the universe become cold, silent, and empty.

## Core Pitch

Long ago, the universe was protected by living light engines called Aether Cores.
These cores powered the Neon Grid, a cosmic network of worlds, sectors, gates,
and ancient machines.

An ancient dead-geometry force called The Null King woke beneath the Grid. It
does not simply destroy worlds. It erases meaning from them. Cities become empty
shapes. Stars become silent. Living things become hostile fragments called the
Swarm.

The player is the last awakened Aether Core, carrying the soul and combat memory
of Nova Veyr.

Nova must fight through corrupted sectors, recover lost Prism Shards, awaken
allies, recover memories, and rebuild the broken Neon Grid before the Null King
turns the universe into a silent black void.

## Player Identity

Nova Veyr was once a legendary Rift Runner who piloted an experimental living
engine called the Aether Core.

During the first Null invasion, Nova sacrificed their body to seal the breach.
Their mind, memory, and fighting instinct were fused inside the Aether Core.

The player is not just a ship. The player is Nova's soul inside a living
geometric weapon.

## World

The Neon Grid is an ancient cosmic infrastructure of light, gates, machine
temples, sector engines, defense rails, memory vaults, and Prism Shard locks.
It once connected living worlds through motion and color. After Lumenfall, the
Grid became fragmented. Some sectors still spark with life. Others are broken
into hostile geometry.

The Swarm is not a simple alien army. It is what remains when the Null King's
erasure breaks matter, memory, and machine logic into weaponized fragments.

The Null King's goal is not conquest. It wants the universe to stop meaning
anything.

## Locked Story Terms

- Aether Core: living light engine and player body, holding Nova Veyr's soul and combat memory.
- Neon Grid: cosmic sector network of worlds, gates, rails, machines, and memory systems.
- The Swarm: hostile fragments produced by Null corruption.
- Null King: ancient dead-geometry force that erases color, motion, memory, life, and meaning.
- Prism Shards: broken memory-lock fragments that can restore the Grid and reveal Mira's fate.
- Neon Dust: purified Swarm residue used to repair and refine memory-code.
- Armory: recovered weapon memory vault where Nova manages known combat forms.
- Forge: Lyra's repair system for refining unstable weapon memory code.
- Run Weapons: temporary combat routines active during the current run.
- Weapon Memories: combat patterns recovered from Nova's past and Grid fragments.
- Rift Runner: old title for pilots who navigated unstable Grid sectors.
- Lumenfall: the first catastrophic Null invasion and collapse of the old light civilization.

## Systems Connected To Story

- XP shards: broken Grid memory.
- Weapons: recovered combat memories.
- Random stats: unstable Core weapon memory variation.
- Forge: Lyra repairing and refining weapon memory code.
- Neon Dust: purified Swarm residue.
- Run Weapons: temporary combat routines that fire during the current run.
- Evolution / Fusion: Nova combining weapon memories into stronger forms.
- Sector events: broken Grid systems trying to reconnect.

## Main Story Arc

1. Nova awakens as the Aether Core inside the broken Neon Grid.
2. Lyra Quill locks onto Nova's signal and begins repairing the Core remotely.
3. The Swarm attacks as corrupted Grid systems come back online.
4. Rook-7 identifies Nova as an old Rift Runner from before Lumenfall.
5. Prism Shards reveal Mira Sol's voice and fragments of Nova's past.
6. Nova learns the Aether Core was not only a defense engine, but a weapon-memory vessel.
7. Hyper Grid reveals that Mira became the living lock holding back the Null King.
8. The Black Crown forces Nova to defeat the Null King without destroying Mira's trapped light.
9. Nova uses the Prism Shards to rebuild the Grid and keep Mira's light alive.
10. The Swarm is pushed back, but the final crown fragment implies darkness can return.
