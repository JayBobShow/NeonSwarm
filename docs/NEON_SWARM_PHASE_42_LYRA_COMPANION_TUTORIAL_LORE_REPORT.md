# Neon Swarm Phase 42 Lyra Companion Tutorial Lore Report

## Scope

Phase 42 adds Lyra Quill as a lightweight in-game companion/tutorial radio
voice. The system is intentionally small: short lines, local runtime seen flags,
automatic fade, and no full cutscene or dialogue framework.

This phase does not start Phase 43, does not build sector story progression,
does not build an ending sequence, and does not change gameplay balance.

## Runtime Location

- Dialogue constants, runtime state, UI creation, queue/fade logic, and triggers
  are in `scripts/NeonSwarm3DGameplayPrototype.gd`.
- The panel node is `LyraQuillCompanionRadioPanel`.
- The speaker label is `LyraCompanionSpeakerLabel`.
- The body label is `LyraCompanionBodyLabel`.

## UI Placement

- Design-space rect: `Rect2(300, 812, 760, 118)`.
- Placement: lower-left / lower-center radio plate.
- It avoids the Phase 38 left stat stack, right eight-slot weapon stack, top
  boss/combat notices, and the center player area.
- It uses a small neon frame, `LYRA QUILL` speaker label, and one short line.

## Lines Added

1. `Nova, you are awake. Good. The Grid is breaking, and you are the only Core still burning.`
2. `Move fast. The Swarm reads hesitation like blood in the water.`
3. `Those shards are broken Grid memory. Collect enough and the Core will adapt.`
4. `Battle data is stabilizing. Pick an upgrade before the Swarm changes its mind.`
5. `That is a run weapon. It starts firing now, but it will not replace your Armory loadout.`
6. `Neon Dust. Purified Swarm residue. Ugly source, beautiful upgrade material.`
7. `Bring enough Dust back and I can rebuild those weapon memories into something nastier.`
8. `Large hostile signature ahead. Nova, that is not a swarm fragment. That is a commander.`
9. `The Grid gate is open. Whatever is beyond it will be worse. Obviously.`
10. `Core integrity is dropping. I need you alive, Nova. Preferably in one piece.`
11. `Signal lost... no, wait. I still have you. Reboot the Core. We go again.`
12. `Sector link restored. That is one light back in the universe.`

## Triggers Implemented

- First gameplay start after the Phase 41 intro or intro skip.
- First movement input during gameplay.
- First XP pickup.
- First level-up / upgrade choice.
- First new run weapon activation.
- First gameplay Neon Dust gain.
- First Forge / weapon-memory reference after the first gameplay Neon Dust gain.
- Boss warning, once per sector warning.
- Sector transition, once per sector transition.
- Low health warning at or below 35% health with an 18-second cooldown.
- Death / run failed screen.
- Sector clear and run victory screens.

No requested triggers were deferred.

## Skip And Fade Behavior

- Lines fade in/out automatically.
- Base display time is 4.8 seconds, with a small duration extension for longer
  lines.
- Cancel input dismisses the active Lyra line when it will not interfere with
  active reward/menu flows.
- Dialogue does not pause gameplay and does not block movement or shooting.

## Audio Behavior

- Adds a subtle original procedural SFX key: `lyra_radio`.
- It is generated by the existing `_make_tone_sfx()` path.
- It uses no external or copyrighted audio.
- It is mixed quietly at `-12.0 dB` before the normal SFX volume multiplier.

## Save Compatibility

- Phase 42 uses local runtime seen flags only.
- No new save/settings keys are added.
- Existing weapon inventory, Neon Dust, Forge, settings, and tutorial save data
  schemas are unchanged.
