## Campaign Pacing Lock

Each active sector should eventually grow from a single pre-boss stretch into:

1. Entry arena
2. Escalation arena
3. Pressure arena
4. Identity arena
5. Boss approach arena
6. Boss fight

Subsectors can carry smaller story beats such as Lyra comments, memory echoes,
enemy faction hints, environmental names, and objective flavor. Major reveals
remain tied to bosses and Memory Shards.

## Runtime Data Decision

No runtime data was added in Phase 46. This keeps the approved four-sector run,
Phase 45 Memory Shard triggers, Phase 44 four-active-boss scope, save data, and
official scene path unchanged.

Future Phase 47 should decide and implement the centralized campaign data shape
before enabling subsector progression.
