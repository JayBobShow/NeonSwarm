# Neon Swarm Memory Shard Bible

## Purpose

Memory Shards are recovered Prism Shards carrying broken Grid memory. They turn
major boss / sector progress into earned story reveals about Nova Veyr, Mira
Sol, the Aether Core, the first Null invasion, and the Neon Grid.

Memory Shards should stay short, emotional, readable, and tied to gameplay
progress. They are not a replacement for the future Lore Codex.

- Future shards must not trigger without a real approved Sector 5 / final boss
  implementation.
- Unlock tracking is runtime-run scoped in Phase 45.

- Memory Shard reveal panels should not block sector rewards or hide critical
  combat information for long.
