# Neon Swarm Reference Image Library

This folder is the official reference image library for Neon Swarm art direction.

Codex must check this folder before doing visual/art/modeling work.

## Reference Types

### user_original_art/
User-owned original artwork, drawings, paintings, tattoo designs, sketches, and visual concepts.

These are project art-direction references provided by the user. Codex may use them as the primary intended visual direction for Neon Swarm.

### user_feedback/
Screenshots from the current game build showing what to keep, reject, or fix.

### sector-specific folders
References for each sector's intended look and readability.

### third_party_inspiration_do_not_copy/
Outside inspiration only.

These images are for studying mood, workflow, lighting, readability, composition, materials, and quality targets.

Do not directly copy third-party copyrighted models, textures, logos, characters, layouts, or designs unless the user confirms they own or have permission to use them.

## Rules

- User-provided original art is allowed as Neon Swarm project reference.
- The user's original art may be used as the intended project art direction.
- Outside inspiration is for learning and direction only, not direct copying.
- For every visual phase, Codex must report which reference folder/images were checked.
- Visual work must still be validated in the actual Godot gameplay camera.
