# Neon Swarm

Neon Swarm is an active-development neon arena survival RPG / roguelite prototype built in Godot.

Official run command:

```bash
godot --path /home/jason/GodotProjects/NeonSwarm scenes/Main.tscn
```

Official scene:

```text
scenes/Main.tscn
```

This project is still in active development. It needs more content, systems, polish, testing, and production work before it should be treated as a finished shareable build.
